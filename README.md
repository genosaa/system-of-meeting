说明：
1.rar后缀为整个项目的压缩档，需要使用的话下载此档即可，其他的为各功能关键代码，供大家线上查阅
2.lib里面的都是相关jar包，有的时候出错是由于里面没有相应jar包，注意
3.使用此项目时，相应的JDBCUtils的相关数据库记得换成你自己相应的数据库哦
